# Microcredit-Loans
Loan Analyzer program.

## Dependencies 

_Python 3.7+_

## Execution Instructions

1. Navigate to the following directory:

```Microcredit-Loans/Code/```

2. Activate _Anaconda_ dev environment.

3. Run the following commmand:

```python loan_analyzer.py```
